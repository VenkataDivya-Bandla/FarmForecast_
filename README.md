# Crop Price Prediction System

## 🚀 Overview
A machine learning-based system that predicts crop and fruit prices using historical and seasonal data to assist farmers in decision-making.

## ✨ Features
- Predicts crop prices based on historical data
- Helps in crop selection and pricing strategies
- Data visualization for insights
- Supports sustainable agriculture planning

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- MongoDB
- MERN Stack (for frontend/backend integration)

```bash
Open index.html in browser
